# Игра "Быки и Коровы"

## Описание игры
Пользователь вводит минимальное и максимальное число для компьютера, чтобы тот загадывал число, 
а затем "игрок" должен угадать его, получая подсказки больше, или же меньше число.

## Правила игры
- Больше - заданное число стоит выше
- Меньше - заданное число стоит ниже

## Требования
- Интерпретатор: Python
